Working Directory

Es el area de trabajo, donde rara vez vamos a estar utilizando la linea de comandos, salvo por el inicio del proyecto o a lo sumo la eliminacion del mismo. Se refiere a la carpeta donde iniciamos el repositorio y basicamente donde vamos a crear, modificar y eliminar los archivos que formen parte de nuestro proyecto.

  • dir || ls muestra el contenido de la carpeta actual
  • mkdir <name> crea una carpeta con el nombre indicado
  • cd <path> abre el directorio especificado
  • git init inicializa un repositorio de git
  • rm -rf .git elimina el repositorio de manera forzada
